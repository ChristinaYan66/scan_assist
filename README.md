# ScanAssist

Photo-based step-by-step AI guidance for everyday devices and appliances.

## Run Locally

```bash
cp .env.example .env
npm start
```

Open `http://localhost:5173`.

## Built With

- Node.js built-in `http`, `fs`, and `path` modules
- Vanilla HTML, CSS, and JavaScript
- OpenAI Responses API
